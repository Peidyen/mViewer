1. You can add Indexes to any collection.
  2. Select a collection you want to add Indexes and click Add/Edit Indexes A Modal is opened as shown in the figure below.

 

  1. The Modal shows all the attributes on which Indexes can be added. You can perform below actions.
  • Select any attribute
  • Select Asc Index for the attribute, if you want to create a ascending index. By default a decending Index is created.

 

  1. Click Submit

 

A Success message is displayed if the Index is successfully added.
